partner-support : Название

Created by Smirnova Olga, last modified by Kosareva Vera Vladimirovna on 30 Mar 2020

Название формируется одним из способов:

  • Составляется автоматически из обязательных характеристик — по умолчанию для всех категорий. Название формируется по схеме: тип + бренд или производитель + модель + важные характеристики для категории.
  • Вы можете задать название вручную (самостоятельно указать название, которое будет на карточке товара). Эти названия попадают на модерацию и должны удовлетворять требованиям. Ручное название доступно в категориях:
    • Печатные книги
    • Электроника → Чехол, Переноска и хранение
    • Ремонт и строительство
    • 18+ (товары для взрослых)
    • Хобби и творчество
    • Зоотовары
    • Детям и мамам
    • Красота и здоровье
    • Антиквариат
    • Спорт и отдых
    • Канцелярия
    • Дом → Мебель
    • Автотовары
    • Продукты питания
    • Аптека
    • Продукт медиа (цифровые товары)
    Если в этих категориях вы не укажете название, оно тоже сформируется автоматически из обязательных характеристик.

Зачем составлять название правильно

Правильное название поможет вашим товарам:

  • Избежать ошибок при создании товара или во время модерации.
  • Появляться на более высоких позициях в результатах поиска Ozon.

  • Выглядеть привлекательнее для покупателя: в результатах поиска и на страницах категорий видна только первая часть названия, а значит, в ней должны быть самые важные параметры.

    Пример

    Полное название: 

    Подушка "Green Line", наполнитель: бамбуковое волокно, полиэстер, 50 х 70 см, белая

Обязательные правила

На что влияют правила

Если название составлено не по этим правилам, товар не будет создан в личном кабинете или позже не пройдет модерацию.

Как составить название

Название:

  • Содержит тип товара (что это за товар, например: кукла, шкаф, смартфон, мусорное ведро).
  • Соответствует товару.
  • Начинается с большой буквы.
  • Можно использовать:
    • русские и английские буквы;
    • цифры;
    • символы: . , : ; ( ) - / & " ;
    • символы ! и ? , если без них название будет некорректным (например, восклицательный знак входит в название книги).
  • Не содержит неразрывных пробелов.
  • Максимальная длина — 255 символов.
  • Грамотное, без ошибок.
  • Не содержит слов, написанных транслитом или большими буквами. Исключения: допускаются аббревиатуры (например, ГОСТ) или названия, которые используют производители (например, SEGA).
  • Не содержит лишней (в том числе, рекламной) информации. Например, нельзя писать:
    • слова «акция», «распродажа», «скидка», «только сегодня»;
    • цену товара;
    • фразы «лучший товар», «уникальный товар», «потрясающий товар» и похожие.

Общие рекомендации

На что влияют рекомендации

Название, составленное по рекомендациям, поможет товарам:

  • появляться на более высоких позициях в результатах поиска Ozon;
  • выглядеть привлекательнее для покупателя: в результатах поиска и на страницах категорий видна только первая часть названия, а значит, в ней должны быть самые важные параметры.

Как составить название

Составьте название по схеме: тип товара (что это) + бренд или производитель (если есть) + название товара (ваше название и важные характеристики, если есть). Примеры:

  • Правильно: Пазл для малышей (тип) Step Puzzle (бренд) Попугай Кеша (ваше название).
  • Неправильно: Step Puzzle Попугай Кеша (нет типа — покупатель не поймет, что это за товар), Пазл для малышей Попугай Кеша (нет бренда), Step Puzzle Пазл для малышей Попугай Кеша (сначала бренд, потом тип).

Для категории Красота и здоровье составьте название по формуле: Бренд или производитель + тип + название модели (если есть) + другие важные параметры (если есть).

Не указывайте слишком много характеристик, оставьте в названии только самые важные.

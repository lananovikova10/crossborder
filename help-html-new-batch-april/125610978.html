partner-support : Статистика по завершенной акции

Created by Kosareva Vera Vladimirovna, last modified by Sosnovskaya Mariya on 11 Mar 2020

Чтобы посмотреть статистику по завершенной акции:

  1. Перейдите в раздел Продвижение → Акции → Завершенные.
  2. На карточке с акцией нажмите Скачать статистику.

На ваш компьютер будет загружен XLS-файл со статистикой.

В файле содержится 3 листа:

  • Описание
  • Результаты акции
  • Данные для сравнения

Описание

Содержит общую информацию по акции:

  • Название акции;
  • Тип акции;
  • Аудитория — категория покупателей Ozon, для которой предназначена акция;
  • Период акции;
  • Условия участия в акции.

Результаты акции

Содержит результаты участия в акции по каждому товару:

  • Цены:
    • Последняя цена продажи до начала акции;
    • Цена по акции;
    • Продолжительность участия товара в акции по такой цене;
    • Период участия товара в акции по такой цене.
  • Продажи:
    • Продано, шт.;
    • Продано, руб.

Если товар участвовал в нескольких акциях одновременно, то он продавался по самой низкой из цен по акции. Если товар был продан по цене из другой акции, в полях Продано, шт. и Продано, руб. будет указан 0 (даже если товар купили со страницы текущей акции).

Результаты участия в акции по одному и тому же товару показаны в нескольких строках, если:

  • в период проведения акции цена по акции (скидка) на товар менялась;
  • товар был исключен из акции, а позже добавлен снова.

Данные для сравнения

Содержит:

  • сравнение результатов участия в акции с показателями за неделю до акции — средней ценой и количеством проданного товара;
  • список более ранних акций, в которых участвовал товар.
